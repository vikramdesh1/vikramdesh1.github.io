Hand Sanitization Reminder System for Hospitals
  • Developed a system which detects when a person enters a room in a hospital and reminds them to sanitize their hands. This ensures that even if the person forgets to sanitize their hands, the system will remind them resulting in reduced transmission of pathogens
  • Technologies used : Texas Instruments MSP-EXP430G2 Launchpad, ElecFreaks HC-SR04 ultrasonic ranging module, Code Compiler Studio, C
